RONDA 2 DE 4

Métricas y aprendizaje

Elige qué mides. Las métricas que eliges determinarán la calidad de tus decisiones futuras.

📌 ¿Qué son las métricas vanidosas?

Las métricas vanidosas hacen que el proyecto parezca exitoso sin revelar si el modelo de negocio funciona. Las métricas accionables te ayudan a decidir si pivotear, optimizar o escalar.

⚠ Advertencia pedagógica Un equipo que elige predominantemente métricas vanidosas puede sobreestimar su desempeño y tomar decisiones equivocadas.

Experimento de validación

¿Cómo validarás que tu propuesta de valor funciona este trimestre? Cada experimento tiene un alcance distinto y responde a un objetivo de negocio específico.

🌐 Landing page con formulario
Alcance: página simple con propuesta de valor y registro de interesados.
Objetivo de negocio: medir intención de uso inicial y claridad del mensaje antes de invertir en desarrollo.
🎯 Piloto con 200 usuarios reales
Alcance: prueba limitada con usuarios reales usando una versión funcional del servicio.
Objetivo de negocio: medir activación, retención y uso efectivo en condiciones cercanas al mercado.
🗣️ Entrevistas en profundidad
Alcance: conversaciones estructuradas con usuarios, clientes o aliados clave.
Objetivo de negocio: comprender dolores, barreras de adopción y razones detrás del abandono o baja conversión.
🧪 Prueba A/B de precios
Alcance: comparar dos niveles de precio, planes o mensajes de monetización.
Objetivo de negocio: validar disposición a pagar y estimar el impacto del precio sobre conversión e ingresos.
🤝 Concierge MVP
Alcance: entregar manualmente el servicio a un grupo pequeño antes de automatizarlo.
Objetivo de negocio: validar valor percibido, flujo de servicio y disposición a pagar con bajo desarrollo tecnológico.
💨 Smoke test de demanda
Alcance: campaña breve con anuncios, correo o lista de espera para medir respuesta del mercado.
Objetivo de negocio: estimar demanda, costo de adquisición y atractivo del segmento antes de lanzar.
🧭 Prueba de onboarding y activación
Alcance: usuarios prueban el primer recorrido, registro o prototipo navegable.
Objetivo de negocio: identificar fricciones tempranas y mejorar la tasa de activación inicial.
📄 Preventa o carta de intención
Alcance: solicitar compromisos, preórdenes o cartas de intención a clientes potenciales.
Objetivo de negocio: validar monetización, urgencia de compra y probabilidad real de cierre comercial.

📖 Ayuda conceptual y fórmulas

Ingresos

Ingresos = Clientes de pago × Precio

Costos totales

Costos totales = Costos fijos + Costos variables + Marketing + Desarrollo + Soporte

Margen de contribución

Margen unitario = Precio − Costo variable unitario

Punto de equilibrio

PE (clientes) = Costos fijos / Margen de contribución unitario

Relación Beneficio / Costo (B/C)

B/C = Valor presente de beneficios / Valor presente de costos B/C > 1 → el proyecto genera más valor del que consume

VAN — Valor Actual Neto

VAN = −Inversión inicial + Σ [ Flujo_t / (1 + r)^t ] r = tasa de descuento, t = período VAN > 0 → el proyecto crea valor

TIR — Tasa Interna de Retorno

TIR = tasa r tal que VAN = 0 TIR > tasa de descuento → proyecto conveniente

PR — Período de Recuperación

PR = período en que los flujos acumulados igualan la inversión inicial Menor PR → recuperación más rápida

VAE — Valor Anual Equivalente

VAE = VAN × [ r(1+r)^n / ((1+r)^n − 1) ] Útil para comparar proyectos de distinta vida útil

IVAN — Índice de VAN

IVAN = VAN / Inversión inicial IVAN > 0 → crea valor por cada peso invertido Útil cuando el capital disponible es limitado

Métricas accionables vs. vanidosas

Las métricas vanidosas (descargas, visitas, registros acumulados) pueden mejorar sin que el negocio mejore. Las métricas accionables (retención, conversión, CAC, margen) revelan si el modelo funciona y guían decisiones.
